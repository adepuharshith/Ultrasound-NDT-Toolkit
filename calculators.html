🧮 Calculators

Interactive tools for common ultrasonic NDE computations — live, no install required

Wave Physics

📈 Wavelength Calculator

Wavelength λ
— mm
λ = c / f

Tip: use c = 5900 m/s for steel (L-wave), 3200 m/s for steel (S-wave), 1480 m/s for water.

Beam Physics

🔭 Near-Field Distance (N)

Near-Field Distance N
— mm
N = D² / (4λ) = D² · f / (4c)

Inspection is most reliable at or beyond N. For focused transducers, focal depth should be ≤ N.

Depth Sizing

⏳ Time-of-Flight → Depth

Depth (one-way)
— mm
d = (TOF × c) / 2

Enter the total round-trip time measured from the A-scan. Result is one-way depth to the reflector.

Signal Loss

🔴 Attenuation Loss

Total Attenuation
— dB
Loss = α × f × d

Typical α: steel 0.3–1.5 dB/mm/MHz, aluminum 0.2–0.8, composites 1–5 (fiber direction), HDPE 2–5.

Refraction & Mode Conversion

📈 Snell's Law & Critical Angle

Refracted Angle θ2
— °
sin(θ2) = (c2 / c1) · sin(θ1)

Default: water → steel (L-wave). Use c2 = 3200 m/s for shear wave refraction in steel. "TIR" = total internal reflection.

Immersion Testing

💧 Water Path Calculator

Must be ≤ material thickness

Required Water Path (WP)
— mm
WP = f − mp × (cs / cw)

Set the transducer face this far above the material surface. The equivalent path length in material (mp × cs/cw) shifts the acoustic focus to the desired depth.

cw default 1495 m/s ≈ water at 23 °C. Adjust for temperature: ~3 m/s per °C.
